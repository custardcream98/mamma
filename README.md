🌊 Wavvie, 맘마먹자!

Wavve 사내에서 관리되고 있는 **대동맛지도** 기반으로 랜덤하게 점심을 추천해주는 서비스입니다.

**[서비스 링크 🔗](https://mamma.vercel.app/)**

> Wavve 임직원을 위한 서비스로, `wavve.com` 도메인으로만 가입할 수 있습니다. 비 로그인 상태에서도 랜덤 추천 기능은 사용할 수 있습니다.
